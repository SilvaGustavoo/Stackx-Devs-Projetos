Escreva um programa para ler 2 valores (considere que não serão informados valores iguais) e escrever o maior deles.

INICIO
x, y: Real

escreva("Digite algum número: ")
leia(x)
escreva("Digite um outro número: ")
leia(y)

se x > y entao 
    escreva("O numero maior é ", x)
senao
    escreva("O numero maior é ", y)
fim_se

FIM
